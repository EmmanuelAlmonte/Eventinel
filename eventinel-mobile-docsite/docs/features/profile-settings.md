---
title: Profile And Settings
description: Stable guide to profile, appearance, push notifications, and account actions.
---

The Profile screen is the central user settings hub.

## Entry point

- Bottom tab: `Profile`

## What users can do

## Identity and account view

- See profile avatar, display name, optional `nip05`, and about text.
- See current public key in short and full forms.

## Appearance

- Toggle dark mode from the Appearance card.
- Theme mode applies immediately across the app.

## Settings links

- Open Wallet screen from the Settings card.
- Open Relay Settings from the Settings card.

## Push notifications

- Check current notification permission state.
- Request notification permission from inside the app.
- Register and persist Expo push token.
- Copy token text from the token panel when present.

## Account actions

- Logout with confirmation prompt.
- Session is removed locally on logout.

## Notes for operators

- Push token registration is most reliable on physical devices.
- If permission is denied, users must re-enable notifications in OS settings.
