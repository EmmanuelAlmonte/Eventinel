---
title: run
description: Run Eventinel on local targets.
---

Use these scripts during day-to-day development.

## Usage

```bash
npm start
```

## Platform targets

| Command | Description |
| --- | --- |
| `npm run android` | Launch Android build/run flow. |
| `npm run ios` | Launch iOS build/run flow. |
| `npm run web` | Start app in browser through Expo. |

## Examples

```bash
npm start
npm run android
npm run ios
```
